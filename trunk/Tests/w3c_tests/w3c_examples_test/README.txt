 Examples

    * examples.xsd - all of the example schemata in a single XML Schema 1.0 document
    * examples2wsdl.xsl - stylesheet to select examples to build a WSDL 1.1 document
    * examples.wsdl - all of the example schemata in a single WSDL 1.1 document
    * examples-soap11.xml - all of the example instance documents as a sequence of SOAP 1.1 messages
    * examples-soap12.xml - all of the example instance documents as a sequence of SOAP 1.2 messages
